# LISCAPS: LSTM-based Intelligent Stock Crawl, Analysis and Prediction System

## LISCAPS：基于 LSTM 的智能股票爬取分析与预测系统

## Usage

1. `.\init_venv.bat`
  Be patient and wait until "Press any key to exit..."
2. `python main.py`

## Environment

- Windows 7+
- Python 3.6+
- numpy
- scipy
- pandas
- matplotlib
- tensorboard
- torch
- visdom
- scikit-learn
- streamlit
- requests
